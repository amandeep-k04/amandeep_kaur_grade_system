# Student Grade Calculator

## Description
This program calculates total marks, average marks, and letter grade for a student using C.  
It is built using multiple files.

## How to Compile
gcc main.c grade.c -o grade_program

## How to Run
./grade_program

## Function Prototype
A function prototype tells the compiler a function exists before it is used.

## Scope
Scope means where a variable can be used.  
Global variables are accessible everywhere.  
Local variables exist only inside functions.

## Why Header Files Are Used
Header files allow multiple files to share function declarations and variables.
